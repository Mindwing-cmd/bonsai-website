# UI Components (shadcn-style)

This folder follows the **shadcn/ui** convention: reusable, copy-paste UI components that depend on `@/lib/utils` and Tailwind. Keeping them in `components/ui` separates design-system primitives (Card, Spotlight, SplineScene) from feature-specific components in `components/`, and makes it easy to add more shadcn or Aceternity components later.
